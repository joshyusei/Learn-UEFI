用 MSYS2 UCRT64 下載 QEMU
pacman -S mingw-w64-ucrt-x86_64-qemu

將 msys64\ucrt64\bin 加入到 PATH 環境變數中

確認 qemu 可以執行
qemu-system-x86_64 -version

安裝 NASM
https://www.nasm.us/
將 nasm/ 加入到 PATH 環境變數中

將 FirstBL.asm 編譯成 FirstBL.bin
nasm -f bin FirstBL.asm -o FirstBL.bin

使用 QEMU 執行 FirstBL.bin
qemu-system-x86_64 -drive format=raw,file=FirstBL.bin
